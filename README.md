# pd-

PD (Pre-Disbursement) Update Verification System

## Stack
- **Frontend**: React + Vite + TailwindCSS → Deployed on Vercel
- **Backend**: Node.js + Express + Supabase → Deployed on Render

## Setup

### Backend
```bash
cd backend
npm install
# Create .env with SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
